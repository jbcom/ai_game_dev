Main files: ['main.py', 'game.py', 'player.py']
Asset requirements: ['player.png', 'background.png', 'sounds.wav']
Build instructions: Build Instructions for Pygame Project:

1. Install Python 3.11+
2. Create virtual environment: python -m venv venv
3. Activate: source venv/bin/activate (Linux/Mac) or venv\Scripts\activate (Windows)
4. Install dependencies: pip install -r requirements.txt
5. Run: python main.py

For distribution:
- PyInstaller: pip install pyinstaller && pyinstaller --onefile main.py
- Auto-py-to-exe: GUI wrapper for PyInstaller
